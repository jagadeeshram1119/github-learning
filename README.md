# Git & GitHub Learning Repository

Welcome! This repository is my **practice space for learning Git and GitHub**.  
It is organized **topic-wise** to make learning easy and track progress.

---

## 📂 Repository Structure

### 01_basics
- `init_repo.md` – Initialize repository
- `git_status.md` – Check file status
- `git_add_commit.md` – Stage and commit changes
- `git_log.md` – View commit history

### 02_branching
- `create_branch.md` – Create new branches
- `switch_branch.md` – Switch between branches
- `merge_branch.md` – Merge branches
- `delete_branch.md` – Delete branches

### 03_remote
- `connect_github.md` – Connect local repo to GitHub
- `push_code.md` – Push code to remote
- `pull_code.md` – Pull changes from remote
- `clone_repo.md` – Clone repository

### 04_advanced
- `stash.md` – Temporarily save changes
- `rebase.md` – Reapply commits on top of another base
- `reset.md` – Undo commits
- `cherry_pick.md` – Apply specific commit from another branch

### practice
- `file1.txt` – Dummy file for practice
- `file2.txt` – Dummy file for practice
- `test_changes.txt` – Test edits and commits

### commands_cheatsheet.md
- All essential Git commands in one place

---

## ✅ How I Use This Repo

1. Create/modify files in topic folders  
2. Practice Git commands on these files  
3. Commit and push changes to GitHub  
4. Track progress with checklists

---

## 📌 Tips for Learning Git

- Always write meaningful commit messages  
- Practice creating and merging branches frequently  
- Try working with remote repos (push, pull, clone)  
- Experiment with stash, reset, rebase, cherry-pick  
- Check GitHub web interface and terminal commands
